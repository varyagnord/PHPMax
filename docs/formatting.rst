Text Formatting
===============

PyMax поддерживает базовое Markdown-подобное форматирование при отправке
сообщений. Форматирование применяется автоматически в ``send_message()``,
``Message.answer()``, ``Message.reply()`` и ``Chat.answer()``: библиотека
удаляет служебные маркеры из текста и передает Max список ``Element``.

.. code-block:: python

   await message.answer(
       "**Важное:** проверьте [документацию](https://example.com)"
   )

Получатель увидит текст без маркеров, а фрагменты будут оформлены средствами
клиента Max.

Поддерживаемый синтаксис
------------------------

.. list-table::
   :header-rows: 1
   :widths: 30 30 40

   * - Синтаксис
     - Тип элемента
     - Описание
   * - ``**текст**``
     - ``STRONG``
     - Жирный текст.
   * - ``_текст_`` или ``*текст*``
     - ``EMPHASIZED``
     - Выделенный текст.
   * - ``__текст__``
     - ``UNDERLINE``
     - Подчеркнутый текст.
   * - ``~~текст~~``
     - ``STRIKETHROUGH``
     - Зачеркнутый текст.
   * - ```код```
     - ``MONOSPACED``
     - Моноширинный фрагмент в строке.
   * - ``[текст](https://example.com)``
     - ``LINK``
     - Ссылка с отображаемым текстом.
   * - ``# Заголовок``
     - ``HEADING``
     - Заголовок в начале строки.
   * - ``> цитата``
     - ``QUOTE``
     - Цитата в начале строки.

Блоки кода
----------

Многострочный код оборачивается в тройные обратные кавычки. Если после
открывающего маркера указать имя языка, оно не попадет в текст сообщения.

.. code-block:: python

   await message.answer(
       """```python
   print("hello")
   ```"""
   )

Этот фрагмент будет отправлен как текст ``print("hello")`` с элементом
``CODE``.

Ограничения
-----------

Это не полный Markdown-парсер. Форматтер поддерживает только перечисленные
маркеры из ``pymax.formatting.markdown.Formatter``.

Обычные inline-маркеры не продолжаются через перенос строки. Если закрывающий
маркер не найден в той же строке или внутри маркеров пустой текст, маркер
остается обычным текстом.

Экранирование служебных символов обратным слешем пока не обрабатывается.
